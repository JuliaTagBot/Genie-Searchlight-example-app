## Instructions: 
1. clone the repo
2. in a terminal, `cd` to the repo's folder
3. start a julia REPL session in the repo's root
4. `julia> ]` to enter Pkg mode
5. `pkg> activate .`
6. `pkg> instantiate` to download the dependencies for the project
